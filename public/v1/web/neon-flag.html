In this project you'll make your first web page!

1. Using HTML, make at least 5 div's in the body of the page.

Tip: It can be helpful to give each div a border for now -- it makes them easier to see.

2. Give each of your div's a unique id.

3. Using CSS, set the size of the div's. Make them all the same size, but make sure they take up the whole room of the page!
